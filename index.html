Catch the Falling Objects Game

Project Outline

An interactive browser game where the player uses a real-world potentiometer connected to an Arduino to move a paddle and catch falling circles. The goal is to catch as many falling objects as possible before missing three.

Narrative Description

The potentiometer’s analog readings are sent from the Arduino to the browser via serial communication. In the browser, P5.js reads the potentiometer data and maps it to the paddle’s horizontal position.

Circles fall from the top of the screen at random positions. Each time the player catches a circle, the score increases. Missing three circles ends the game.
